# Personal Site

A minimal blog built with [Astro](https://astro.build).

## Local Development

```bash
npm install
npm run dev
```

## Adding Posts

Create a markdown file in `src/content/posts/`:

```markdown
---
title: Your Title
subtitle: Optional subtitle
date: 2024-11-25
type: reading  # or: listening, thinking
tags:
  - books
  - fiction
---

Your content here.
```
